Need to know
Common questions
🔑
How do I get a Replicate API key?
Go to replicate.com/account/api-tokens, sign up free, and create a token. You'll get some free credits to start — most images cost a few cents each.
🔒
Is my API key safe?
Your key is stored only in your browser session memory and is only sent directly to api.replicate.com. It's never sent to PixelBoost or any third-party server.
💰
How much does each upscale cost?
The philz1337x/crystal-upscaler model on Replicate is billed per second of GPU time. A typical upscale costs $0.01–$0.05 depending on image size and scale factor.
⏱️
How long does it take?
Most images process in 15–60 seconds on Replicate's GPU servers. Larger images and higher scale factors (6×) take longer. The progress bar updates in real time.
